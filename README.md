# Universal Pokemon Randomizer ZX - Redux Edition

This is a fork of the Universal Pokemon Randomizer ZX (UPR-ZX) designed to support modern ROM hacks, specifically tailored for **Pokemon Blaze Black 2 & Volt White 2 Redux**.

## Supported ROM Hacks
- **Pokemon Blaze Black 2 Redux / Volt White 2 Redux** (Complete support for Version 1.6+)

## WHAT WORKS FOR SURE :
- **Starter Pokemon Randomization**
- **Gift Pokemon Randomization**: Supports Redux-exclusive gifts randomization including:
  - Elemental Monkeys (Floccesy Town)
  - Gen 1 Starters (Virbank City)
  - Gen 2 Starters (Castelia City)
  - Gen 3 Starters (Anville Town)
  - Gen 4 Starters (Driftveil City)
  - Eevee (Castelia City)
  - Corsola (Season Research Lab)
- **Static Pokemon**: unverified but should work
- **Bypassed Hack ROM Warnings**: No more "Unrecognized ROM Hack" errors, hardcoded though.

##In progress:
-Eggs (i.e first egg of redux in the ranch).
-Might crash after rival first battle, very unlikely though but occured once.

## How to Use
1.  **Download** the latest release or clone the repo
2.  **Run**: launcher for your OS.
3.  **Randomize**: Select your Redux ROM and choose your desired settings!
